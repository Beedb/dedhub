Ded Hub HTML é um hub de estudos front-end que centraliza recursos gratuitos de HTML, CSS e JavaScript, organizados por nível, tipo de conteúdo e idioma.

---

## Sobre o projeto

O objetivo é fornecer um ambiente simples e eficiente para estudar front-end, reunindo conteúdos confiáveis em uma interface organizada e intuitiva.

Os materiais são filtráveis e categorizados, permitindo uma progressão estruturada no aprendizado.

---

## Funcionalidades

- Organização por nível (iniciante, intermediário, avançado)  
- Filtros por tipo de conteúdo e idioma  
- Navegação por abas (HTML, CSS, Ferramentas e Dúvidas)  
- Dark mode com persistência via localStorage  
- Interface responsiva  
- Renderização dinâmica dos conteúdos  

---

## Tecnologias

- HTML5  
- CSS3 (variáveis, responsividade)  
- JavaScript (DOM, filtros dinâmicos)  
- Google Fonts  
- Font Awesome  

---

## Estrutura de aprendizado

1. HTML (estrutura)  
2. CSS (estilização)  
3. Layout (Flexbox)  
4. Prática e resolução de dúvidas
